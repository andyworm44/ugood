# 🧪 UGood 應用測試指南

## 📋 **測試檢查清單**

### 🎤 **錄音功能測試**

#### **基本錄音測試**
- [ ] **權限測試**
  - 首次使用時是否正確請求麥克風權限
  - 拒絕權限後的錯誤處理
  - 重新授權權限的流程

- [ ] **錄音品質測試**
  - 在安靜環境下錄音
  - 在嘈雜環境下錄音
  - 測試不同音量大小
  - 測試長時間錄音（接近3分鐘）

- [ ] **錄音控制測試**
  - 開始錄音按鈕響應
  - 停止錄音按鈕響應
  - 錄音計時器準確性
  - 3分鐘自動停止功能

#### **播放功能測試**
- [ ] **立即播放測試**
  - 錄音完成後立即播放
  - 播放進度顯示
  - 播放/暫停控制
  - 播放完成自動停止

- [ ] **音質測試**
  - 播放音質是否清晰
  - 是否有雜音或失真
  - 音量是否適中
  - 播放時長是否正確

### 🔄 **完整流程測試**

#### **自我配對流程**
1. [ ] 進入錄音頁面
2. [ ] 錄製一段祝福語音（建議內容）：
   ```
   "你好！我想告訴你，雖然生活有時候很困難，
   但是每個挫折都是成長的機會。
   相信自己，你一定可以度過難關。
   記住，你並不孤單，總有人在默默支持你。加油！"
   ```
3. [ ] 測試錄音播放功能
4. [ ] 發送祝福語音
5. [ ] 選擇"聆聽祝福"
6. [ ] 在聆聽頁面播放自己的錄音
7. [ ] 確認UI顯示"自己的錄音"標識

#### **模擬祝福流程**
1. [ ] 從首頁點擊"🎧 體驗模擬祝福"
2. [ ] 測試模擬播放功能

## 🚀 **測試建議**

### **測試環境設定**
1. **使用真實設備**：模擬器無法測試錄音功能
2. **不同網路環境**：WiFi、4G、5G、弱訊號
3. **不同時間測試**：白天、晚上、嘈雜環境

### **測試內容建議**
```
測試錄音內容範例：
1. "這是一個測試錄音，1、2、3、4、5"
2. "Hello, this is an English test recording"
3. "今天天氣很好，我想分享一些正能量"
4. "長篇測試：生活就像一盒巧克力，你永遠不知道下一顆是什麼味道..."
```
